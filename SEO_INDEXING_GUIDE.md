## Common Indexing Issues & Solutions

### If pages aren't indexing after 2 weeks:

1. **Check for blocking:**
   - Verify no meta robots="noindex"
   - Check .htaccess for redirects
   - Ensure pages return 200 status code

2. **Content quality:**
   - Ensure sufficient unique content (500+ words)
   - Avoid duplicate content
   - Add images and multimedia

3. **Internal linking:**
   - Link to new pages from high-authority pages
   - Add breadcrumb navigation
   - Create contextual links in related content
