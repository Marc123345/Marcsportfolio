# Marc Friedman Portfolio

A modern, high-performance portfolio website showcasing design and development services.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Supabase (Backend & Auth)
- Framer Motion
- GSAP

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run sitemap` - Generate sitemap
- `npm run check:indexing` - Check for SEO/indexing issues
- `npm run seo:check` - Run full SEO check

## Environment Variables

See `.env.example` for required environment variables.
